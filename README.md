# Графический Калькулятор на C (EasyX) #

Простой интерактивный калькулятор с графическим интерфейсом, написанный на языке C с использованием библиотеки EasyX. Проект демонстрирует работу с графическими примитивами, обработку событий мыши и клавиатуры, а также параметрическую верстку интерфейса.

🛠 Технологический стек
 - Язык: C (*Visual Studio Code*)
 - Графическая библиотека: [EasyX Graphics Library](https://easyx.cn/)
 - ОС: Windows

 📂 Структура проекта
    main.c — основной цикл программы, обработка сообщений и логика вычислений.
    constants.h — конфигурационный файл с настройками цветов, размеров и констант интерфейса.
    .gitignore — настроен на игнорирование временных файлов VS Code и исполняемых файлов .exe

⚙️ Установка и запуск
1) Установите EasyX: Скачайте и установите библиотеку с официального сайта easyx.cn.
2) Клонирование проекта
3) Компиляция 
<pre>
g++ start.cpp -o calculator -leasyx
</pre>
